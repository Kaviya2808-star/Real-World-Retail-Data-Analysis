# Real-World Retail Data Analysis Using the Superstore Dataset

## Project Overview
This project analyzes the Superstore Retail dataset using Python. The objective is to clean the data, perform exploratory data analysis (EDA), create visualizations, and build a Linear Regression model for sales prediction.

## Dataset
- Dataset: Superstore Retail Dataset
- File Name: Super_store.csv

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Import the dataset.
2. Explore the dataset.
3. Clean the data.
4. Perform Exploratory Data Analysis (EDA).
5. Create visualizations.
6. Build a Linear Regression model.
7. Evaluate the model using MAE, MSE, and R² Score.

## Visualizations
- Sales by Category
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Top 10 Products by Sales

## Machine Learning Model
- Algorithm: Linear Regression
- Features: Quantity, Discount, Profit
- Target: Sales

## Model Evaluation
- Mean Absolute Error (MAE): 245.08
- Mean Squared Error (MSE): 680377.46
- R² Score: -0.152

## Conclusion
The project successfully demonstrates retail data analysis using Python. Data cleaning, visualization, and a machine learning model were implemented to gain business insights. Future improvements can be made by using additional features and more advanced machine learning algorithms.
